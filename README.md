# Локальная кольцевая «почта» (вариант 18)

Программа передачи сообщений между тремя узлами в кольцевой топологии с реализацией **физического**, **канального** и **прикладного** уровней.

## Быстрый старт (демо на одном ноутбуке)

```bash
cd "/Users/acti0n/Documents/Курсач"
python3 -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
python3 demo.py
```

Откроются **три окна** (узлы 1, 2, 3). Канал подключается автоматически.

### Сценарий демонстрации

1. В окне **узла 1**: меню **Почта → Написать письмо** → адресат **узел 2** → отправить.
2. В окне **узла 2**: во **Входящих** двойной щелчок по письму → **Отметить прочитанным**.
3. В окне **узла 1**: во **Исходящих** статус: «доставлено» → «прочитано».
4. Внизу каждого окна — **протокол событий** (передача, ACK, ошибки CRC).

## Архитектура

| Уровень | Каталог | Функции |
|---------|---------|---------|
| Физический | `src/physical/` | Виртуальные COM, параметры (скорость 8N1), чтение/запись по двум портам |
| Канальный | `src/datalink/` | Кадры HDLC, CRC-16-CCITT, маршрутизация по кольцу, ACK доставки/прочтения |
| Прикладной | `src/application/` | Письма, входящие/исходящие, уведомление о прочтении |
| GUI | `src/ui/` | Окна входящих/исходящих, журнал, настройки COM |

## Запуск отдельных узлов

Три терминала с общим симулятором (на одной машине):

```bash
python3 main.py --node 1 --demo-ring
python3 main.py --node 2 --demo-ring
python3 main.py --node 3 --demo-ring
```

С реальными портами (Windows, com0com):

```bash
python main.py --node 1 --serial-left COM3 --serial-right COM4
```

## Формат кадра

`FLAG | DST | SRC | TYPE | HOP | SEQ | LEN | PAYLOAD | CRC16 | FLAG`

Типы: DATA, ACK_DELIVERED, ACK_READ, ERROR.

## Требования

- Python 3.10+
- tkinter (обычно встроен в Python)
- pyserial (для реальных COM; в демо не обязателен)
